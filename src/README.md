# Анализатор папок и бэкапов
Консольная утилита для сканирования папок, поиска дубликатов по MD5 и верификации резервных копий с защитой от системных ошибок доступа.

## Запуск
bash
python main.py <folder_path> [<backup_folder>]

## Аргументы
Folder_path	Путь к основной папке для сканирования
backup_folder	Путь к папке с резервной копией (необязательно)

## Структура проекта
main.py — точка входа, обработка аргументов командной строки
scanner.py — рекурсивный обход папок, фильтрация файлов и форматирование вывода
dublicat.py — подсчёт MD5-хэша файлов и группировка дубликатов
check_backup.py — логика пофайлового сравнения оригинала с резервной копией

## Фильтрация
По умолчанию программа автоматически пропускает скрытые файлы/папки (начинающиеся с точки, например .git, .idea), файлы нулевого размера, а также файлы, к которым нет доступа.

## Дубликаты
Программа рассчитывает хэш-сумму MD5 для каждого файла. Если у нескольких файлов (даже с разными именами) совпадает хэш, утилита выводит их в консоль как дубликаты с указанием путей к ним.

## Резервная копия
При указании пути к бэкапу программа строит карту относительных путей и сравнивает содержимое оригинальной папки с резервной копией, разделяя различия на три категории:

Отсутствует в бэкапе — файлы, которые есть в исходной папке, но не скопированы
Лишний в бэкапе — файлы, которые существуют в бэкапе, но уже удалены или отсутствуют в оригинале
Изменён — файлы, которые есть в обоих местах, но их MD5-хэши отличаются (содержимое перезаписано)
